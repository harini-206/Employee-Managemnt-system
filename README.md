# Employee-Managemnt-system

A web-based Employee Management System built using Flask that enables organizations to efficiently manage employees, departments, attendance, and leave requests. The application implements secure authentication, role-based access control, and a user-friendly dashboard for administrators and employees.

🚀 Features

* 🔐 User Authentication (Admin & Employee)
* 👥 Employee Management (Create, Read, Update, Delete)
* 🏢 Department Management
* 🗓️ Attendance Management
* 📝 Leave Request & Approval System
* 📊 Dashboard with Statistics
* 🔍 Employee Search & Filters
* 👤 Profile Management
* 📁 File Upload Support (Profile Photos)
* 📄 Pagination for Employee Records
* 🛡️ Role-Based Access Control

 🎯 Learning Objectives

This project demonstrates practical implementation of:

* Full-stack web development with Flask
* Database design and relationships
* Authentication and authorization
* CRUD operations
* Form validation
* File handling
* Clean project architecture using Blueprints
